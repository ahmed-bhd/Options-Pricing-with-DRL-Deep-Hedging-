# Options-Pricing-with-DRL-Deep-Hedging-
This project implements **Deep Hedging** - using Deep Reinforcement Learning (DRL) to optimally hedge European call options in the presence of transaction costs and discrete trading constraints. The system compares classical hedging strategies (Black-Scholes Delta, Delta-Gamma, Static hedges) against state-of-the-art DRL agents (PPO, SAC, TD3).
